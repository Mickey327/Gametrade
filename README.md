# Gametrade
 Сайт-магазин внутриигровых ценностей. Курсовая работа за 3 семестр.
 Реализовано 6 страниц: главная, каталог, faq, о компании, пользовательское соглашение, политика конфиденциальности.
 Сайт сделан с применением технологий JS, HTML5, CSS3, также были использованы библиотека JQuery для JS и библиотека Bootstrap для HTML5, CSS3, JS.
